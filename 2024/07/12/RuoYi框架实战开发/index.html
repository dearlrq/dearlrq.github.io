4.9.3 全局异常处理器，捕获并处理异常

  • SQLntegrityConstraintViolationException：这个类通常用于表示 SQL 数据库操作中的完整性约束违反异常。

  • 例如：外键约束、唯一约束等。当数据库操作违反了这些约束时，就会抛出这个异常。

  • DatalntegrityViolationException：在使用 SpringBoot 框架时，如果违反了数据完整性抛出这个异常。

  • dkd-framework \ web \ exception \ GlobalExceptionHandler
